## Executive Summary

이별방패는 **국내 유일의 데이트폭력 전문 법률 앱**으로, 명확한 블루오션에 위치합니다. AI 사무장 + 변호사 최종 결정 구조는 법적으로 안전하며, 370만 건 판례 DB와 결합하면 강력한 경쟁우위를 확보할 수 있습니다.

### 핵심 수치
- **TAM**: 1,200~1,600억원 / **SOM 3년차**: 15~30억원
- **권장 가격**: 월 2,900원 (Phase 1) → 2,900/4,900원 2-tier (Phase 2)
- **개발 비용**: Phase 1~3 합계 약 1.7~2.5억원
- **BEP**: MAU 15,000~20,000명 (출시 9~10개월차 예상)

## 2. AI 사무장 시스템 — "공감하는 최고의 사무장"

### 역할 정의
AI 사무장은 피해자와 변호사 사이의 **중간 계층**으로, 5가지 핵심 기능 수행:

| 기능 | 출력물 |
|------|--------|
| 감정 위로 및 공감 | 공감 메시지, 안전 확인 |
| 사실관계 파악 (5W1H) | 사실관계 요약서 (JSON) |
| 증거 수집 가이드 | 증거 체크리스트 |
| 법적 쟁점 정리 (RAG) | 쟁점 분석서 + 판례 3~5건 |
| 변호사 선택지 제시 | 옵션 A/B 비교표 |

### 5단계 대화 플로우

```
Phase 1: 안전 확인 → "지금 안전한 곳에 계신가요?"
Phase 2: 상황 파악 → 5W1H 기반 구조화 수집
Phase 3: 증거 확인 → 보유 증거 체크 + 추가 수집 안내
Phase 4: 법률 분석 → 370만 DB RAG → 판례/법조항 매칭
Phase 5: 선택지 생성 → 변호사용 2개 옵션 + 근거 판례
```

### 변호사 의사결정 화면 (예시)
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 사건 요약서 #2026-0323-001
의뢰인: 김OO (여, 28세)
상황: 이별 후 3개월간 일 50회 전화, 주거지 방문 2회
쟁점: 스토킹처벌법 적용, 접근금지 가처분

옵션 A: 형사 고소 우선 (신속 보호, 보복 위험)
옵션 B: 민사 가처분 + 경고 (단계적, 시간 소요)

AI 의견: 위험도 HIGH — 옵션 A 권고

     [옵션 A 선택]  [옵션 B 선택]  [수정 요청]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 감정 분석 & 위기 감지
- **5차원 감정 분석**: 공포/불안/분노/무력감/자기비난
- **3단계 위기 프로토콜**: 주의 → 경고(자해 언급) → 긴급(즉각 위험)
- 자기비난 감지 시: "절대 본인의 잘못이 아닙니다" 자동 응답

#### Phase 2 (7~12개월): 2-Tier 구조

- **케어플랜 4,900원/월** — 변호사 상담 포함 + 법률문서

#### 이 가격이 작동하는 이유
1. 2,900원은 "돈 때문에 고민"을 사실상 제거
2. AI 사무장이 1차 처리 → 초저가에도 원가 구조 성립
3. 구독은 "입구", **변호사 수임 연결 + B2B/B2G가 실질 수익원**
4. 정부 보조금(여가부, 법무부) + ESG 투자 유치 가능

## 4. UX/브랜딩 — "전문가가 지키고 있다는 확신"

### Trust Signal 3원칙
> 모든 화면에서 항상 보여줘야 할 것:
> 1. **누가** 담당하는지 (김창희 변호사 프로필 상시 노출)
> 2. **지금** 어떤 단계인지 (진행 상태 바)
> 3. **다음에** 무엇이 일어나는지 (다음 단계 안내)

### 브랜드 톤 3T
- **T**ender (다정): "많이 힘드셨죠. 잘 오셨어요."
- **T**rustworthy (믿음직): "법적으로 보호받을 수 있는 방법이 있습니다."
- **T**ough (단호): "상대방의 행위는 스토킹처벌법 위반에 해당합니다."

### Trauma-Informed Design 핵심
- 터치 타겟: 56x56dp (떨리는 손 고려)
- 한 화면에 질문 1개만
- 선택형(탭) 우선, 타이핑 최소화
- "지금은 여기까지만 하고 싶어요" 항상 제공

### 사용자 감정 여정
```
온보딩 → AI대화 → 증거수집 → 변호사검토 → 법적조치 → 사후관리
불안/공포 → 안도 → 능동감 → 신뢰/기대 → 안심 → 회복
```

### 은밀모드 고도화
- 앱 아이콘: 계산기/날씨/메모 선택 가능
- 알림 위장: "내일 날씨: 맑음" (실제는 사례 업데이트)
- 볼륨 3회 = 즉시 위장 화면 전환
- PIN 5회 오류 = 자동 로컬 데이터 삭제

## 5. 기술 아키텍처 — "370만 건 DB × RAG"

### 시스템 구성도
```
React Native (Expo)
       │
       ▼
Supabase Edge Functions (Deno)
  ├── 대화 관리 → Claude API (대화 엔진)
  ├── RAG 파이프라인 → pgvector (370만 건)
  └── 감정 분석 → 위기 감지
       │
       ▼
Supabase PostgreSQL
  ├── court_cases (판례 280만 건) + pgvector
  ├── statutes (법조항 50만 건)
  ├── legal_qa (법률 Q&A 40만 건)
  └── case_chunks (판례 청크 ~3,000만 건)
```

### 하이브리드 검색 (벡터 + 키워드)
1. 사용자 쿼리 → 의미 임베딩 + 키워드 추출
2. 시맨틱 검색 (cosine) + 키워드 검색 (ts_rank) 병렬
3. RRF 결합 → 상위 5~10건 → Claude 컨텍스트 주입
4. Hallucination 방지: Citation 강제 + DB 존재 검증

### Hallucination 방지 4중 장치
1. **Grounded Generation**: 검색 결과에 근거해서만 답변
2. **Citation 강제**: 모든 법적 주장에 [판례번호] 의무
3. **사후 검증**: 언급된 사건번호를 DB에서 실조회
4. **Confidence 폴백**: 유사도 < 0.75 → "변호사 직접 상담 권합니다"

### 구현 로드맵 & 비용
| Phase | 기간 | 핵심 산출물 | 비용 |
|-------|------|-----------|------|
| **1. MVP** | 12주 | AI 사무장 + RAG + 모바일 앱 | 8,000~12,000만원 |
| **2. 대시보드** | 8주 | 변호사 웹 대시보드 + 결제 | 5,000~7,000만원 |

| **합계** | **28주** | | **1.7~2.5억원** |

## 6. 법적 리스크 — "안전한 구조, 주의할 포인트"

|--------|--------|------|

| Claude API 데이터 국외이전 | 🔴 높음 | DPA 체결 + 비식별화 + 동의 획득 |

| 승소율 표방 금지 | 🟡 중간 | AI 출력 필터링, "참고 판례"로만 표현 |
| 민감정보 동의 | 🟡 중간 | 별도 동의 절차 정비 |
| AI 표시의무 | 🟢 낮음 | UI에 고지 문구 상시 표시 |

### 안전한 서비스 구조
```
AI 사무장 = "변호사의 내부 업무 보조 도구"
─ AI가 사실관계 정리 (비법률 영역)
─ 변호사가 선택지 확정 (법률 판단)
─ IT 파트너에게는 고정 용역비만 지급
─ 모든 AI 출력에 "검토: 김창희 변호사" 표시
```

## 결론

이별방패는 **"여성 변호사가 직접 만들어주는 데이트폭력 방지 앱"**이라는 유일무이한 포지셔닝을 가지고 있습니다.

**3가지 핵심 경쟁우위:**
1. **370만 건 판례 DB + AI RAG** — 어떤 경쟁자도 즉시 따라올 수 없는 데이터 해자
2. **변호사 직접 운영** — 변협 규제 회피 + 후발주자 진입장벽
3. **초저가 구독(2,900원)** — "돈 때문에 고민"을 제거하는 사회적 가치

**즉시 실행 가능한 Next Step:**
1. Supabase pgvector 셋업 + 370만 건 임베딩 시작
2. AI 사무장 System Prompt 파일럿 테스트
