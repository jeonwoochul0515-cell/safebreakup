## 1. 서비스 개요

### 이별방패란?

데이트폭력·스토킹·디지털성범죄 피해자를 위한 **여성 변호사가 직접 운영하는 법률 보호 앱**입니다.

AI 법률 사무장이 따뜻하게 공감하며 상황을 파악하고, 370만 건 판례 데이터베이스에서 관련 사례를 찾아, 김창희 변호사가 최종 법률 판단을 내리는 구조입니다.

### 핵심 차별점

| 기존 서비스 | 이별방패 |
|------------|---------|
| 로톡 등 중개 플랫폼 (변협 규제 리스크) | **변호사가 직접 운영** (규제 안전) |
| 범용 법률 플랫폼 | **데이트폭력 전문 특화** |
| 사람이 1:1 상담 (비용 높음) | **AI 사무장 + 변호사** (월 2,900원) |
| 긴급 시 대응 어려움 | **SOS + 은밀모드 + 24시간 AI** |

### 타겟 사용자

- 20~30대 여성
- 데이트폭력/스토킹/디지털성범죄 피해자 또는 이별을 준비 중인 사용자
- 극도로 불안한 심리 상태, 가해자의 핸드폰 검사 가능성 있음
- 모바일 90%+ 접속

### 4.2 AI 사무장 대화 (`app/ai-secretary.tsx`)

**핵심 기능** — 5단계 대화형 법률 상담 시스템

```
Phase 1: 안전확인 → "지금 안전한 곳에 계신가요?"
Phase 2: 상황파악 → 5W1H 구조화 질문 (선택형)
Phase 3: 증거확인 → 증거 체크리스트 (다중선택)
Phase 4: 법률분석 → 적용 법률 + 유사 판례 표시
Phase 5: 선택지생성 → 옵션 A/B 카드 (변호사 결정용)
```

**주요 기능:**
- 감정 분석: 사용자 메시지에서 위기 키워드 자동 감지
- 위기 배너: "죽고싶", "자살" 등 감지 시 112/1366 긴급 배너 즉시 표시
- 일시정지: "지금은 여기까지만 하고 싶어요" 버튼 항상 표시
- 변호사 검토: 모든 법률 안내에 "검토: 김창희 변호사" 표시
- 타이핑 인디케이터: 500ms 지연 후 봇 응답 (생각하고 있다는 느낌)

| 위장 모드 | 외관 | 해제 방법 |
|----------|------|----------|
| **계산기** | iOS 스타일 계산기 | PIN "1234" 입력 후 "=" |
| **날씨** | 날씨 앱 (서울, 18°) | 검색창에 "1234" 입력 |
| **메모** | 메모 앱 (가짜 메모 목록) | 4번째 메모 길게 누르기 |

#### 안전 설정 (`app/safety-settings.tsx`)
- 위장 아이콘 선택 (계산기/날씨/메모)
- 알림 위장: "내일 날씨: 맑음" (실제는 앱 알림)
- 빠른 탈출: 볼륨 3회 / 흔들기 / 사용 안함
- PIN 변경 (기본 1234)
- PIN 실패 시 자동 데이터 삭제 (3/5/10회)
- 원터치 전체 데이터 삭제 버튼

## 5. 안전 기능 체계

### 5.1 플로팅 SOS 버튼

|--------|------|------|

| 디지털성범죄센터 | 02-735-8994 | 디지털 피해 지원 |

### 5.3 위기 감지 시스템
AI 사무장 대화 중 키워드 기반 감정 분석:
- **위기(crisis)**: 죽고싶, 자살, 자해 → 즉시 긴급 배너
- **고통(distressed)**: 무서워, 두려워, 힘들어 → 공감 응대 강화
- **불안(anxious)**: 불안, 걱정, 어떡해 → 안심 메시지
- **안정(calm)**: 정상 대화 진행

### 6.2 타이포그래피
- 폰트: Noto Sans KR (시스템 폰트 폴백)
- 크기: xs(11) → sm(13) → md(15) → lg(17) → xl(21) → xxl(26) → hero(34)

### 6.3 간격
- 4px 기반: xs(4) → sm(8) → md(16) → lg(24) → xl(32) → xxl(48) → xxxl(64)

### 6.4 모서리 곡선
- sm(10) → md(14) → lg(20) → xl(28) → full(9999)

### 6.5 그림자
- 울트라 소프트: 넓은 반경(8~24px), 낮은 불투명도(4~8%)

### 6.6 디자인 원칙
1. **Trauma-Informed Design**: 최소 터치 타겟 56dp, 한 화면 한 질문, 선택형 우선
2. **Trust Signal**: 모든 화면에 변호사 프로필/검토뱃지/진행단계 표시
3. **톤 3T**: Tender(다정) + Trustworthy(믿음직) + Tough(단호)
4. **안전 최우선**: 은밀모드, SOS는 어떤 화면에서든 접근 가능
