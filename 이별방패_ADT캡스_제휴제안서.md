## 2. 시장 기회

### 핵심 데이터

- **연간 스토킹 신고**: 120,000+건 (2024)
- **피해자 76.2% 여성**, 78.7%가 20-30대
- **1인 가구 740만** (2025), 여성 1인가구 빠르게 증가
- **데이트폭력 검거**: 매일 37명 (2024)
- **디지털 성범죄 피해자**: 10,305명 (2024)

### 핵심 인사이트

> **법률 보호와 물리적 보안을 결합한 서비스는 시장에 전무합니다.**

스토킹·데이트폭력 피해자는 법률 대응과 물리적 안전 확보를 동시에 필요로 하지만, 현재 시장에는 이를 원스톱으로 제공하는 서비스가 존재하지 않습니다. 이별방패와 ADT캡스의 제휴는 이 시장 공백을 최초로 채우는 기회입니다.

## 3. 이별방패 소개

### 서비스 개요

- **운영**: 법률사무소 청송 (대표변호사 김창희) 직접 운영
- **규모**: 39개 화면, 8개 핵심 모듈
- **타겟**: 20-30대 여성

### 핵심 모듈

| 모듈 | 설명 |
|------|------|
| AI 법률사무장 | 24시간 법률 상담 AI |
| 위험도 진단 | 관계 위험도 실시간 평가 |
| 증거 포렌식 | 디지털 증거 수집·보전 |
| 스토킹 대응 | 스토킹처벌법 기반 대응 체계 |
| 디지털 성범죄 대응 | 불법촬영·유포 대응 |
| 이별 안전 경호 | 이별 통보 시 전문 경호 서비스 |
| SOS 긴급 시스템 | 원버튼 긴급 신고 + 위치 공유 |
| 안전 체크인 | 정기 안전 확인 시스템 |

## 4. ADT캡스에게의 가치

### 신규 시장 진입

- **2030 여성 세그먼트**: 캡스 주고객(40-50대) 대비 새로운 고객층 확보
- **1인 가구 여성 보안 시장 선점**: 급성장하는 여성 1인가구 시장 조기 진입

### 브랜드 이미지 강화

- **ESG/CSR**: 여성 안전 사회공헌 활동으로 기업 이미지 제고
- **언론 노출**: 스토킹·데이트폭력 등 사회적 이슈와 연계한 자연스러운 미디어 노출

### 매출 기회

- **이별방패 유저 → 캡스 가입 전환**: 예상 전환율 5-10%
- **안심팩 공동상품**: 월 34,800원 ~ 89,800원 신규 매출

## 5. 제안 모델

### 모델 A: API 연동

| 항목 | 내용 |
|------|------|
| **연동 방식** | 이별방패 SOS → 캡스 출동 API 연결 |
| **자동 통보** | 안전 체크인 미응답 → 캡스 관제센터 자동 통보 |
| **수익 구조** | 출동 건당 수수료 또는 월정액 제휴비 |
| **구현 기간** | 2-3개월 |

**기술 연동 흐름**:
1. 이별방패 앱 SOS 버튼 → GPS 좌표 + 사용자 정보 전송
2. ADT캡스 관제센터 API 수신
3. 가장 가까운 출동 요원 자동 배치
4. 출동 상태 실시간 앱 피드백

### 모델 B: 공동 상품 (안심팩)

**3개 티어 구성**:

| 티어 | 월 가격 | 포함 서비스 |
|------|---------|------------|
| **베이직** | 34,800원 | 이별방패 케어 + 캡스 긴급출동 |
| **스탠다드** | 54,800원 | 베이직 + CCTV + 스마트 도어락 |
| **프리미엄** | 89,800원 | 올인원 + 야간순찰 + 신변보호 |

**수익 배분 (예시)**: 이별방패 20% / ADT캡스 80%

### 모델 C: 마케팅 제휴

- **캡스 고객 → 이별방패**: 할인 코드 제공
- **이별방패 고객 → 캡스**: 설치비 무료 혜택
- **공동 마케팅 캠페인**: SNS, 뉴스 등 미디어 협업
